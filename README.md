# ContractShield AI

> **YOUR TRUSTED LEGAL AID**
> 
> AI-powered legal intelligence that explains complex contracts in plain English, highlights risky clauses, and helps you sign with confidence.

---

## 🚀 Key Features

*   **Single Contract Analysis**: Drag and drop any agreement (PDF or DOCX) to get a full risk scan instantly.
*   **Executive Dashboard**: Get a quick overview of Contract Type, overall Risk Level, Risky Clause Count, and Key Obligations.
*   **Segmented Risk Needle Meter**: View your contract's overall score on a 3-zone visual risk meter (Low, Medium, High).
*   **Detailed Clause Breakdown**: Review clause details with dynamic action badges (`Negotiate`, `Needs Attention`, `Review`, `Accept`) and clean cards with color-coded severity bars.
*   **AI checklist suggestions**: Modern recommendations with checkboxes for fast scans.
*   **Side-by-Side Comparison**: Upload a draft and standard playbook template to highlight deviations and match obligations.
*   **Interactive Ask Assistant**: A legal-trained Q&A chat assistant grounded in the document text to answer specific contract questions.
*   **Premium PDF Reports**: Clean, professional client-side generated reports using jsPDF, featuring metadata cover pages, table checklists, glossary sheets, and disclaimers.

---

## 🛠️ Technology Stack

*   **Frontend**: HTML5, Vanilla CSS3 (Custom Variables, glassmorphism UI rules, dark/light theme support), Vanilla JavaScript.
*   **Libraries**: jsPDF (for client-side PDF document compilation).
*   **Backend**: Flask (Python web server).

## 📦 Local Setup & Installation

### 1. Prerequisites
Ensure you have **Python 3.x** installed.

### 2. Clone the Repository
```bash
git clone https://github.com/anjika-25/Contract-Shield.git
cd Contract-Shield
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the Development Server
```bash
python app.py
```
Open your browser and navigate to `http://127.0.0.1:5000` to start analyzing.
